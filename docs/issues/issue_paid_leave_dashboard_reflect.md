# 有給承認予定のダッシュボード反映

ステータス: In Progress
作成日: 2025-11-08
担当: 未割当
関連:

## 目的

承認済み有給予定が管理者ダッシュボード（打刻アラート・今日の勤務等）に正しく反映されるようにする。

## 要件

- 有給申請が承認された時点で、該当日が「勤務予定」や「アラート」等に反映されること
- 管理者・一般ユーザー双方のダッシュボードで有給予定が視認できること

## 受入基準

- 承認済み有給が「今日の勤務」「打刻アラート」等に正しく表示される
- 予定反映のタイミング・ロジックが明確である

## API/DB 案

- 有給予定取得 API の拡張、またはダッシュボード集約 API で有給予定も返却

## 備考

- 既存の有給申請・承認フローとの整合性に注意
- 実装時はパフォーマンス・権限・UI/UX も考慮すること
